# Contest Radar Roadmap

## Vision
✅ Contest Radar platform for competitive programming contests
- Simplify contest discovery and participation
- Build community-driven leaderboard and user progress tracking  
- Ensure reliable notifications across devices and platforms
- Scale to support global users with fast, SEO-friendly performance

## Core Features
- ✅ Contest Display: Basic UI structure ready
- 🔄 User Registration & Authentication: Dependencies installed, implementation pending
- ⏳ Alerts & Notifications: Planned for Phase 3
- ✅ User Dashboard: UI structure with placeholder data
- ✅ Leaderboard: Basic page structure ready
- ⏳ Additional Features: Contest reminders, analytics, dark mode

## Tech Stack
- ✅ Frontend: Next.js (React-based) for SSR, routing, and API integration
- ✅ Backend: Next.js API routes (Node.js) prepared
- 🔄 Database: PostgreSQL (with Prisma) - schema ready, connection pending
- 🔄 Authentication: NextAuth.js - dependencies installed, implementation pending
- ⏳ APIs & Scraping: Axios for official APIs; Puppeteer for scraping
- ⏳ Notifications: Browser Notifications API (extension); Nodemailer for emails
- ⏳ Extension: Web Extensions API (Manifest V3)
- ✅ Deployment: Vercel ready for hosting
- ⏳ Other Tools: Tailwind CSS ✅, Jest for testing ⏳, Socket.io ⏳
